# How to Remove Gujarati Iframe from Main Dashboard

## Steps to Edit Dashboard in HA

### 1. Go to Home Assistant Dashboard
- Open: `http://localhost:8123/`
- Navigate to your Family Hub dashboard

### 2. Click the Menu (⋯) Button
- Top-right corner of the dashboard
- Select "Edit Dashboard"

### 3. Find the Gujarati Learning Card
Look for the large card that says **"Gujarati Learn Hub"** with the yellow/purple gradient background
- This is the iframe we want to remove
- It should be near the bottom of the dashboard

### 4. Delete the Gujarati Iframe Card
- Click on the Gujarati card to select it
- Click the delete/trash icon (🗑️) that appears
- Confirm deletion

### 5. Verify the Button is Still There
- Scroll up/down to find the **"📚 Open Gujarati Learning"** button
- This should still be visible (we kept this for parents)
- It should be in the section near other action buttons

### 6. Save the Dashboard
- Click the "Save" button (top-right)
- Or press Ctrl+S
- Wait for confirmation

### 7. Verify Changes
- Dashboard should now show:
  - ✅ "📚 Open Gujarati Learning" button (for parents)
  - ❌ NO full Gujarati iframe taking up space
  - ✅ All other sections intact

---

## What You're Removing vs Keeping

### ❌ **REMOVE** (Full Gujarati Learning Hub iframe)
- Large card with yellow/purple gradient
- Takes up ~8 rows of space
- Shows full learning interface
- Labeled "Gujarati Learn Hub"

### ✅ **KEEP** (Button for parents)
- Small button card
- Says "📚 Open Gujarati Learning"
- Opens Gujarati module when clicked
- Minimal space on dashboard
