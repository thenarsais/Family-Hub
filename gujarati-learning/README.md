# Gujarati Learning Module

Interactive learning module for Krish and Karishma with 158 lessons across:
- **Phase 1:** Alphabet (48 letters)
- **Phase 2:** Numbers (30 numerals 0-100)
- **Phase 3:** Vocabulary (80 words across 6 categories)

## Features
- Learn, Quiz, and Tracing modes
- Text-to-speech pronunciation
- Letter tracing with visual guides
- Progress tracking per user
- Curriculum verified against LearnGujarati.net & Shabdkosh.com

## Files
- gujarati-curriculum-final-verified.json - Complete curriculum data
- gujarati-learning-module.html - Interactive learning interface

## Usage
Open gujarati-learning-module.html in any modern browser. Works standalone or embed in Home Assistant.
